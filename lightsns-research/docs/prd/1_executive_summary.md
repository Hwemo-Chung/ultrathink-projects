## 🎯 프로젝트 개요

**프로젝트명:** LightSNS
**태그라인:** "Connect Everyone, Everywhere"
**카테고리:** Social Networking Platform

### 한 줄 요약
전 세계 22억 저속 인터넷 사용자를 위한 초경량, 초고속 소셜 네트워크 플랫폼

### 핵심 가치 제안

```
기존 SNS        →  LightSNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
250MB 앱        →  15MB (94% ↓)
30초 로딩       →  2-3초 (90% ↓)
300MB/월 데이터 →  30MB (90% ↓)
오프라인 불가   →  완전 지원 ✓
```

### 핵심 문제점

#### 1. 기존 SNS의 한계
**문제:**
- 대용량 앱 크기 (Instagram: 250MB, Facebook: 400MB)
- 느린 로딩 시간 (저속 환경에서 1-2분)
- 과도한 데이터 사용 (월 300-500MB)
- 오프라인 모드 부재

**영향:**
- 22억 명의 사용자 배제
- 높은 진입 장벽 (데이터 비용)
- 열악한 사용자 경험
- 디지털 격차 심화

#### 2. "Lite" 앱의 한계
**기존 Lite 앱의 문제:**
- 기능 제한적 (기본 앱의 60-70%)
- 여전히 높은 데이터 사용량
- 제한적인 오프라인 지원
- 불완전한 현지화

**시장 갭:**
- 진정한 "저속 최적화" 앱 부재
- 1Mbps 이하 환경에 특화된 솔루션 없음

#### 3. 사용자 페인 포인트

**조사 결과 (인도, 케냐, 인도네시아, 1,000명)**

| 문제 | 비율 | 심각도 |
|------|------|--------|
| 앱 다운로드 시간 너무 김 | 87% | 🔴 높음 |
| 로딩 시간 답답함 | 92% | 🔴 높음 |
| 데이터 비용 부담 | 78% | 🔴 높음 |
| 오프라인 사용 불가 | 65% | 🟡 중간 |
| 배터리 빨리 소모 | 56% | 🟡 중간 |

## 💡 솔루션

### LightSNS 접근법

#### 설계 철학: "Mobile-First → Light-First"

```
전통적 접근         LightSNS 접근
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 모든 기능 추가   1. 핵심 기능만
2. 데스크톱 우선    2. 저속 모바일 우선
3. 최적화 (나중)    3. 최적화 (처음부터)
4. 화려한 UI        4. 효율적 UI
```

### 핵심 솔루션

#### 1. 극한 경량화
```yaml
앱 크기 최적화:
  총 크기: 15MB
  구성:
    - 핵심 코드: 5MB
    - UI 컴포넌트: 3MB
    - 이미지 에셋: 3MB
    - 기타: 4MB

비교:
  - Instagram: 250MB (17배 차이)
  - Facebook Lite: 2MB (하지만 기능 제한)
  - LightSNS: 15MB ✅ 최적 균형
```

#### 2. 지능형 데이터 관리
```yaml
데이터 사용량:
  일일: 1-3MB
  주간: 7-21MB
  월간: 30MB

최적화 기법:
  - 이미지 압축: WebP, 저해상도 프리뷰
  - 비디오: 비활성화 또는 초저화질
  - 텍스트 압축: Gzip, Brotli
  - 차등 동기화: 변경분만 전송
```

#### 3. 오프라인 우선 설계
```yaml
오프라인 기능:
  읽기:
    - 피드 (최근 50개 게시물)
    - 프로필 (캐시된 데이터)
    - 메시지 (최근 대화)

  쓰기:
    - 게시물 작성 (큐잉)
    - 댓글 (큐잉)
    - 좋아요 (로컬 저장)

  동기화:
    - 연결 시 자동 동기화
    - 우선순위 기반 전송
    - 충돌 해결 메커니즘
```

#### 4. 점진적 로딩
```yaml
로딩 전략:
  1단계 (0.5초): 텍스트 콘텐츠
  2단계 (1.5초): 저해상도 이미지
  3단계 (3초): 고해상도 이미지 (선택)
  4단계 (백그라운드): 비필수 데이터

목표:
  - 초기 인터랙션: 0.5초
  - 완전 로딩: 3초
  - 체감 속도: "즉시"
```

## ⚙️ 핵심 기능

### MVP 기능 (Phase 1)

#### 1. 사용자 관리
```yaml
기능:
  - 회원가입/로그인 (전화번호, 이메일)
  - 프로필 설정 (이름, 사진, 소개)
  - 친구 추가/관리
  - 알림 설정

최적화:
  - 프로필 사진: 50KB 이하
  - 소개: 150자 제한
  - 로컬 캐싱
```

#### 2. 콘텐츠 생성
```yaml
게시물:
  - 텍스트 게시물 (최대 500자)
  - 이미지 (1장, 200KB 이하)
  - 위치 태그
  - 해시태그

최적화:
  - 자동 이미지 압축
  - 오프라인 작성
  - 연결 시 자동 업로드
```

#### 3. 피드
```yaml
기능:
  - 타임라인 피드
  - 친구 게시물
  - 좋아요/댓글
  - 공유

최적화:
  - 무한 스크롤 (10개씩)
  - 이미지 지연 로딩
  - 텍스트 우선 로딩
```

#### 4. 메시징
```yaml
기능:
  - 1:1 채팅
  - 텍스트 메시지
  - 이미지 공유 (압축)
  - 읽음 표시

최적화:
  - 오프라인 메시지 큐잉
  - 자동 재전송
  - 압축 전송
```

#### 5. 검색
```yaml
기능:
  - 사용자 검색
  - 해시태그 검색
  - 위치 검색

최적화:
  - 로컬 캐시 우선
  - 점진적 결과 로딩
```
